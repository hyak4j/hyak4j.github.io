測試環境

  • Windows 10 家用版

軟體資訊

  • 軟體名稱: RaiDrive
  • 軟體性質: 免費軟體 (含付費版)
  • 介面語系: 支援多國語系 (含繁體中文)
  • 官方網站: https://www.raidrive.com/

安装步驟

1.至官方網站點選下載

2.下載目前最新版本

目前最新版本: 2021.5.20 (檔案大小80.31MB)

3.點選安裝檔 => 開始安裝步驟

raidrive-2021-5-20.exe

同意許可和條件 => 點選安裝

Windows安全性詢問: 確認安裝

4.RaiDrive語系設定

點擊 setting

選取中文(繁體)

5.選取欲掛載之網路硬碟

點擊新增

選取欲掛載之網路硬碟 (以Google Drive為例) 以初始設定即可 => 點選確認

6.登入Google帳密，並將權限授予RaiDrive

7.完成! 本機路徑多了所設定掛載的網路硬碟
